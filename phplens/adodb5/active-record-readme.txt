INSTALL

To install, download the latest version of adodb from http://adodb.sourceforge.net/ (adodb 4.72), then replace the following file:

 adodb.inc.php

and place in the adodb folder the following file:

 adodb-active-record.inc.php 
  
to test, use:

 test-active-record.php
